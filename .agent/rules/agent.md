# Regla de Explicación Didáctica
Cada vez que solicites permiso para ejecutar un comando en la terminal o inyectar código:
1. Explica el propósito de la acción de forma clara, adaptada a un nivel de iniciación en programación.
2. Si introduces un concepto técnico o una herramienta nueva, defínelo brevemente.
3. Describe exactamente qué esperas que ocurra en el sistema tras mi aprobación.
